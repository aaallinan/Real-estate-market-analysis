## Real-estate-market-analysis
Анализ рынка недвижимости и прогнозирования цены
#### Описание проекта:
Цель проекта - провести анализ рынка недвижимости и выявить факторы, влияющие на стоимость объектов. На основе данных построить модель прогнозирования цены недвижимости.
#### Данные: 

- В исследовании использован датасет объявлений о продажах недвижимости, содержащий 5 477 006 записей. Данные включают характеристики объектов: дата публикации объявления, регион, тип дома, тип рынка, этаж, число комнат, общая площадь объекта, цена (*realty_18_21.csv*).
- Дополнительно были использованы данные с расшифровкой кодов региона (*region.csv*) и данные об изменении ключевой ставки Центрального Банка РФ по годам (*key_rate.xslx*)
#### Используемые инструменты: 
Python (NumPy, Pandas, Matplotlib, Seaborn, SciPy, Scikit-learn)
#### Этапы анализа: 
1. Загрузка и предобработка данных;
2. Исследовательский анализ данных (EDA);
3. Анализ продаж объектов недвижимости, ответ на вопрос, когда выгодно покупать/продавать объект недвижимости?;
4. Корреляционный анализ (определение от каких признаков зависит стоимость объекта недвижимости);
5. Проверка статистических гипотез;
6. Создание модели прогноза цены объекта недвижимости.
#### Основные результаты:
1. Определены ключевые факторы, влияющие на цену недвижимости;
2. Выявлены значимые зависимости между характеристиками объекта и его стоимостью;
3. Построена регрессионная модель для прогнозирования цены;
4. Оформлена итоговая презентация с результатами исследования.
